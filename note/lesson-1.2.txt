css - каскадные таблицы стилей
css3 - текущая версия

синтаксис: 
body {
    background-color: black;
}

способы подключения:
через <link>
прописать через <style>
отдельно использовать style для элемента

способы подключения имеют приоритет. 
самый приоритетный это указать в самом элементе.
пример: 
<body style="background-color: teal">

далее по приоритету идет указание в style head. 
пример: 
<head>
        <meta charset="utf-8">
        <title>My first page</title>
        <style type="text/css">
            body {
                background-color: Teal;
            }
        </style>
</head>

через ссылку. пример:
<link rel="stylesheet" href="style.css">

селекторы - специальным образом записанные выражения, позволяющие применить правила форматирования некоторой группе элементов веб-страницы:
селектор {
    правила форматирования ...
}

селекторы дают возможность форматирования сразу группы элементов

селекторы бывают нескольких видов:
селектор элемента
    пример: 
    p {
        color: black;
    }

селектор class. класс используется, когда нужно задать разные стили для одного тега:
    p.first / .first {
        color: white;
    }
    ... <p class="first>Blabla</p>
.first - универсальный класс, не привязанный к конкретному типу элемента. классы можно комбинировать
